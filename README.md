# Fazlı Soluk - Siber Terminal

Bu proje, Termux ortamında siber güvenlik araştırmaları ve araçları için tasarlanmış etkileşimli bir terminal arayüzüdür. Kullanıcılara siber temalı bir deneyim sunarken, temel güvenlik araçlarına hızlı erişim sağlamayı amaçlar.

## Özellikler

- **Siber Temalı Arayüz:** ASCII sanatı ve özel komut istemi ile siber bir atmosfer sunar.
- **Temel Komutlar:** Ağ taraması, güvenlik açığı istismarı ve şifre çözme gibi temel siber güvenlik işlemlerini simüle eder.
- **Genişletilebilirlik:** Kolayca yeni komutlar ve araçlar eklenebilir.

## Kurulum

1. Termux uygulamasını indirin ve kurun.
2. Aşağıdaki komutları Termux terminalinde çalıştırın:

```bash
git clone https://github.com/Fazli4343/FazliSoluk.git
cd FazliSoluk
chmod +x fazlisoluk.sh
./fazlisoluk.sh
```

## Kullanım

Terminal açıldığında, `help` komutunu kullanarak mevcut komutları görebilirsiniz. Örnek komutlar:

- `scan 192.168.1.1`
- `exploit CVE-2023-1234`
- `decrypt sifreler.txt`
- `info`
- `exit`
